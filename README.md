# An Vuong

**Nothing attracts me more than things people say are “impossible.”**

Today, I launched a project that I’ve been really passionate about. I call it **“An Vuong.”**

## Why "An Vuong"?

These represent my two priorities. In Vietnamese:  
- **“An”** (or **“An Tâm”**) means maintaining a peaceful and calm mind.  
- **“Vương”** represents the spirit of fighting or striving forward.  

Together, **“An Vuong”** can be understood as continuing to fight and move forward while keeping a calm and positive mind.

---

## About the Project

**An Vuong** is intended to be a Vietnamese programming language whose syntax is written in Vietnamese.  

I’ve spent years learning programming languages such as Python, Java, and C, all of which require writing in English. One day, while brushing my teeth, I asked myself:  

*"Why don’t I create one that people in Vietnam can learn and understand more easily?"*  

And that’s how I decided to start this project.

## Roadmap

Phase 1 — Build the Lexer
- Step 1: Define token types (TokenTypes.java)
- Step 2: Define Token class (Token.java)
- Step 3: Build the Lexer engine (Lexer.java)
- Step 4: Test it
  
Phase 3 — Parser and AST
- Step 1: Learn what an AST is
- Step 2: Define AST node types
- Step 3: Build the Parser
- Step 4: Test it

Phase 4 — Semantic Analysis
- Step 1: Build a symbol table
- Step 2: Implement scope resolution
- Step 3: Check for errors
- Step 4: Test it
  
Phase 5 — Interpreter
- Step 1: Choose execution strategy
- Step 2: Build the evaluator
- Step 3: Handle variables and scope
- Step 4: Handle functions
- Step 5: Handle runtime errors

Phase 6 — Standard Library
- Step 1: Input and output
- Step 2: String operations
- Step 3: Math operations
- Step 4: Collections
